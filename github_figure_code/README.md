# Ca II K Polar Network Index Figure Code

This folder contains a GitHub-ready subset of the figure code and files for:

**Ca II K Polar Network Index of the Sun: A Proxy for Historical Polar Magnetic Field**  
The Astrophysical Journal, 982:78 (2025)  
DOI: https://doi.org/10.3847/1538-4357/adb3a8

## Contents

- `notebooks/ca_ii_k_pni_paper_figures.ipynb` - renamed and reduced from `theo_koso_data.ipynb`; only published-journal figure cells are kept.
- `figures/published/` - published figure exports, renamed by journal figure number.
- `data/saved_states/` - required `.sav` inputs for the retained figure cells.
- `data/text_outputs/` - required PNI text tables.
- `data/dataverse_files/` - required external data tables.
- `styles/` - Matplotlib style file used by the notebook.
